#  Frictional Business App

A Frictional Business App with contact-us page.

- [Click here](https://incredible-beignet-76a976.netlify.app/) to see preview of my project.

## Tech Stack

- HTML 5
- CSS 3
- Javascript
- React JS
- chakra UI

## Features

- Landing Page
- Contact Us

## Run Locally

Clone the project

```bash
  git clone https://github.com/Manoranjan-Sethi/business-website-insaid.git
```

Go to folder

run the frontend
```bash
 cd frontend
```

Install packages

```bash
 npm i
```

Run the app

```bash
  npm start
```

Go to folder

run the backend

```bash
 cd backend
```

Install packages

```bash
 npm i
```

Run the app

```bash
  npm start
```
