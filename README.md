## Run Locally

Clone the project

```bash
  git clone https://github.com/Manoranjan-Sethi/H-L-blogsWithFirebase.git
```

Go to folder
run the frontend
```bash
 cd frontend
```

Install packages

```bash
 npm i
```

Run the app

```bash
  npm start
```

Go to folder

run the backend

```bash
 cd backend
```

Install packages

```bash
 npm i
```

Run the app

```bash
  npm start
```
